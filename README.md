# Errors Library

Detailed, pretty-printed colorful errors for Go.

Check out the `test/` directory for examples.

```
────────────────────────────────
 [!] Internal Error
    in nested_file.txt:0:2
    └─ from /tmp/some/path/to/a/file.txt:6:9

    5 |  Hello World
    6 |  testing ←—

 >>> This is an example error body
```
